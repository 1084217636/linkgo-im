# Docs

`docs/` 目录存放面向简历和面试的项目材料，不是运行时代码，而是帮助你把项目讲清楚、写清楚。

## 文件说明

- `resume-project.md`：适合放进简历的项目描述。
- `resume-ai-game.md`：面向游戏 AI / AI 工程化岗位的简历版本。
- `im-design-boundary.md`：IM 项目的设计边界、可靠性和面试追问说明。
- `golang-intern-fit.md`：面向 Golang 实习岗位的项目对齐说明。
- `devops-guide.md`：Docker、Kubernetes、CI/CD 的使用说明。
- `project-structure.md`：项目目录结构和文件归属约定。
- `interview-script.md`：适合面试时口头讲解的版本。
- `teaching-guide.md`：从零理解这个升级版项目的教学型总结。
- `learning-map.md`：按服务、链路和技术栈拆开的学习路线图。
